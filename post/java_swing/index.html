模式

组件三要素

  • 内容
  • 外观
  • 行为

模型-视图-控制器(MVC)

  • 模型: 存储内容
  • 视图: 显示内容
  • 控制器: 处理输入

布局管理

面板默认是流布局(flow layout manager). 居中, 从上到下, 满了换行

  • setLayout() 设置布局管理器
  • Component add(Component c)
  • Component add(Component c, Object constraints) 添加组件. constraints 是布局管理器可以理解的标置

边框布局(BorderLayout)

JFrame 内容窗格默认布局器. 分为 5 个部分东、南、西、北、中. frame.add(component, BorderLayout.SOURCE)添加到北, 默认是中(CENTER). 边框布局会扩展组件大小来添充区域

网格布局(GridLayout)

从下到上, 从左到右, 需要指定行数和列数

文本输入

  • 文本域(JTextField): 只接收单行
  • 文本区(JTextArea): 可接收多行
  • 密码(JPassword): 接收单行, 但不显示

其超类 JTextComponent 是抽象类, 其中几个有用的方法

  • String getText()
  • void setText()
  • boolean isEditable()
  • void setEditable(boolean b)

JTextField

常常将其加入面板或其它组件

1
2
3

JPanel panel = new JPanel();
JTextField text = new JTextField("Default", 20);
panel.add(text);

标签和标签组件

  • JLabel(String text)
  • JLabel(Icon icon)
  • JLabel(String text, int align)
  • JLabel(String text, Icon icon, int align)
  • String getText();
  • void setText();
  • Icon getIcon();
  • void setIcon(Icon icon);

密码域(JPasswordField)

  • JPasswordField(String text, int column)
  • void setEchoChar(char c)
  • char[] getPassword(): 安全起见, 使用后应覆盖该数组

文本区(JTextArea)

行数和列数. 为了避免输入过长被截断, 开启换行特性

滚动窗格(JScrollPannel)

将组件放入滚动窗格可以为组件添加滚动条
