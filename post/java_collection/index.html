public class ArrayQueue<E> implements Queue<E>
{
    private int head;
    private int tail;
    private int size;
    private int capacity;
    private E[] elements;
    ArrayQueue(int capacity)
    {
        head = -1;
        tail = -1;
        size = 0;
        this.capacity = capacity;
        elements = (E[]) new Object[capacity];
    }
    public void add(E element)
    {
        if(size == capacity){
            throw new QueueIsFull();
        }
        tail = (tail + 1) % capacity;
        elements[tail] = element;
        size++;
    }
    public E remove()
    {
        if(size == 0){
            throw new QueueIsEmpty();
        }
        int h = head;
        head = (head + 1) % capacity;
        size--;
        return elements[h];
    }
    public int size()
    {
        return size;
    }
}

使用的时候可以使用接口类型存放集合的引用, 这样当想要更换实现时只需更改构造器

集合框架中的接口

  • Collection
  • Map

一些区别

  • Collection 使用 boolean add(E element) 添加元素. Map 使用 V put(K key, V value);
  • Collection 可以使用迭代器访问元素, Map 使用 V get(K key)

具体的集合

集合类型 描述
ArrayList 可以动态增长和缩减的索引序列
LinkedList 可以在任何位置快速插入和删除的 有序 序列
ArrayDeque 用循环数组实现的双端队列
HashSet 无序集
TreeSet 有序集
EnumSet 包含枚举类型的集
LinkedHashSet 记住元素插入顺序的集
PriorityQueue 高效删除最小元素的集
HashMap 存储 key/value 关联的数据结构
TreeMap 键值有序排列的映射
EnumMap 键值无枚举类型的映射
LinkedHashMap 记住元素插入顺序的映射
WeekHashMap 可以回收不用的值
IdentityHashMap 使用 == 而不是 equals 比较的映射

链表

  1. Java 中的链表都是双向链表
  2. add 方法插入末尾
  3. 迭代器 Iterator 的子接口 ListIterator 中 add 方法可以插入当前迭代器之前.
  4. 该子接口也允许反向遍历链表
    1. E previous();
    2. boolean hasPrevious();
  5. remove 删除刚由迭代器访问的元素

数组列表

  • ArrayList 是不同步的
  • Vector 是同步的

散列集

  1. 可以快速查找的数据结构散列表(hash table)
  2. Java 中散列表用链表数组实现

树集

  1. 有序集合
  2. 目前的实现是红黑树

双端队列

  1. Dqueu 接口
  2. ArrayDeque 和 LinkedList 都实现了该接口

优先级队列

堆, 可以快速查找最小值

映射

  1. HashMap
  2. TreeMap

枚举集与映射

枚举集内部用位序表明实现. 没有公共构造器, 可以使用工厂方法

视图与包装器

使用视图(views)可以获取其它实现了 Collection 和 Map 的对象. 视图引用的还是原对象. 可以理解为限制了接口的原数据

集合包装器

  1. Array 的 asList 可以返回一个普通 Java 数组的 List 包装器
  2. Collections 类实现了很多实用方法. 参数和返回值是集合
    1. nCopies(n, anObject) 返回一个实现了 List 接口的不可变对象
    2. singleton(anObject) 返回实现了 Set 的单元素集, 对应的有 singletonList, singletonMap
    3. emptySet() 返回空集

子范围(subrange)

  1. subList(begIndex, endIndex) 可以进行对应的任何操作
  2. 有序集合和有序映射可以按照排序了选子范围
    1. subSet(form, to)
    2. headSet(to)
    3. tailSet(from)
    4. 映射类似
