import java.awt.*;
import javax.swing.*;

import sun.awt.EventQueueItem;
class SimpleFrame extends JFrame{
    private static final int DEFAULT_WIDTH = 300;
    private static final int DEFAULT_Height = 200;
    public SimpleFrame(){
        setSize(DEFAULT_WIDTH, DEFAULT_Height);
    }

    public static void main(String[]args){
        EventQueue.invokeLater(()->{
            SimpleFrame frame = new SimpleFrame();
            frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
            frame.setVisible(true);
            });
    }
}

所有的 Swing 组件应由事件分派线程(event dispatch thread)进行配置

框架定位

  • setLocation setBounds 设置位置
  • setIconImage 设置图标
  • setTitle 设置标题栏文字
  • setResizable 是否允许用户改变大小

setLocation(x,y) 设置位置, 左上角是(0,0) setBounds(x,y,width,height)设置位置和大小. setLocationByPlatform(true)让窗口系统控制窗口的位置

框架属性

属性是获取/设置方法对. getXxx/setXxx, 返回值是 boolean 时获取用 is 而不是 get

获取屏幕大小

为了使组件大小合适, 需要根据屏幕分辩率设置组件大小. 调用 Toolkit 的静态类 getDefaultToolkit 得到一个 Toolkit 对象 getScreenSize 可以返回一个 Dimension

使用颜色

setPaint 后序操作的颜色 Color 类用于定义颜色, java.awt.Color 中定义了 13 个常量

  • BLACK 黑
  • BLUE 蓝
  • CYAN 青
  • DARK_GRAY 黑灰
  • GRAY 灰
  • LIGHT_GRAY 亮灰
  • MAGENTA 品红
  • ORANGE 橙
  • PINK 粉
  • RED 红
  • WHITE 白
  • YELLOW 黄

Color(r,g,b) 可以定义任意颜色
