JAR 文件

Java 归档文件, 包含类文件, 资源文件等

创建 JAR 文件

1

jar cvf JARFilename file1 file2 ...

清单文件

每个 JAR 还包含一个描述归档特征的清单文件, 位于 JAR 特殊目录 MATE-INF 中的 MANIFEST.MF 简单的例子只包含版本

1

MainFest-Version: 1.0

可执行 JAR 文件

可以使用 jar 的 e 选项指定程序入口

或者在清单文件中

1

Main-Class: com.mycompany.myapp.mainclass

资源文件

  • 图象, 声音. 使用 URL 存储位置, 调用 getImage(), getAudioClip
1
2

URL url = ResourceTest.class.getResource("img.gif");
Image img = new ImageIcon(url).getImage();
  • 文本文件
1
2

InputStream stream = ResourceTest.class.getResourceAsStream("text.txt");
Scanner in = new Scanner(stream, "UTF-8");

密封

为了保证没有其它类加入其中, 在清单文件的主节中加入

1

Sealed: true

可以全局设置, 对某一个包可以单独加入该包所在的节

服务加载器

定义一个接口或超类, 包含服务需要提供的接口. 具体的实现类可以放在任意包中, 每个实现必须有无参数构造器把类名增加到 META-INF/services 目录下的 UTF-8 编码文本文件中, 文件名与类名一致

applet

省略

Java Web Start

发布

  1. 打包
  2. 创建 Java Network Launch Protocol(JNLP)格式的描述符文件
  3. 将文件放置在 Web 服务器上
