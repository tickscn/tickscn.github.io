数值类型

  • 标量(Scalar): 单个的实数, 维度为 0. shape 为[]
  • 向量(Vector): n 个实数的有序集合. shape 为[n]
  • 矩阵(Matrix): n 行 m 列实数的有序集合, 维度为 2. shape 为[n, m]
  • 张量(Tensor): 维度大于 2 的数组. 张量的每个维度也被称为轴(Axis)

Tensorflow 中为了方便将标量,向量和矩阵也称为张量

1

Tensorflow 中创建张量的方式为 ~tf.constant()~

字符串类型

通过 tf.constant() 传入字符串对象, 可以创建字符串类型的张量

布尔类型

传入 Python 中的 Boolean 的数据

1

~a = tf.constant(True)~

与 Python 原生的 Bollean 数值比较不相等

数值精度

由数据的字长来控制.

1

精度和类型转换可以使用 ~tf.cast()~

待优化张量

为了区分需要计算梯度信息的张量和不需要计算梯度信息的张量, Tensorflow 有专门的类型来表示需要训练的数据 tf.Variable, 可以由 tf.Variable 来创建, 参数可以是普通 tf 张量, 也可以是原始数组

创建张量

可以通过 numpy.array 或 python 中的切片来创建. tf.convert_to_tensor. numpy.array 默认使用 float64 精度, 如果想的话可以转换为 float32. 实际上可以使用 tf.constant

0 张量或全 1 张量

  1. tf.zeors(): 全 0
  2. tf.ones(): 全 1
  3. tf.zeros_like 与参数同样大小的全 0
  4. tf.ones_like 与参数同样大小的全 1

全指定数值张量

使用 tf.fill(shape, value)

已知分布的向量

可以创建采样自已知分布的随机变量, 常见的有正态分布(Normal Distribution, 或高斯分布 Gauss Distribution)和均匀分布(Uniform Distribution)

  1. tf.random.normal(shape, mean=, stddev=): 创建形状为 shape, 期望为 mean, 标准差为 stddev 的正态分布 \(\mathcal{N}(mean,stddev^{2})\)
  2. tf.random.uniform(shape,mival=,maxval=,dtype=)

创建序列

tf.range(limits, delta=), 序列不包括 limits. 默认为 delta 为 1 的整形序列
