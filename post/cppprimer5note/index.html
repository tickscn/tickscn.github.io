标准异常 (Standard Exceptions)

四个头文件

  • exception 定义了最一般的异常类, 名字是 exception
  • stdexcept 定义了一些多功能的异常
    类名 描述
    exception 最一般的异常类
    runtime_error 只能在运行中发现的错误
    range_error Runtime error: 算术运算结果超出范围
    overflow_error Runtime error: 溢出
    underflow_error Runtime erroe: 下溢
    logic_error 程序的逻辑错误
    domain_error Logic error: 定义域错误
    invalid_argument Logic error: 参数错误
    length_error Logic error: 创建一个对象过大
    out_of_range Logic error: 使用的值超出范围
  • new 定义了 bad_alloc
  • type_info 定义了 bad_cast

#include <iostream>
#include <stdexcept>
int main()
{
    double dividend, divisor;
    while (std::cin >> dividend >> divisor)
    {
        try
        {
            if (divisor == 0) throw std::invalid_argument("除数不能为0");
            std::cout << dividend << " / " << divisor << " = " << dividend / divisor << "\n";
        }
        catch (std::invalid_argument err)
        {
            std::cout << err.what() << "\n 重新输入除数?(y/n)" << std::endl;
            char c;
            std::cin >> c;
            if (!std::cin || c != 'y') break;
        }
    }
    return 0;
}
