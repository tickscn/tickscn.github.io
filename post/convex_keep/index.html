交集

如果 \(S_1, S_2\) 是凸的, 那么 \(S_1\cap S_2\) 也是凸的

仿射函数

一个函数可以表示为线性函数和常数的和, 称该函数为仿射的. 如果集合 \(S\) 是凸的, 函数 \(f\) 是仿射的, \(S\) 在 \(f\) 下的象是凸的 \[f(S) = \{f(\boldsymbol{x})\arrowvert \boldsymbol{x}\in S\}\]

  1. 两个简单的仿射变换: 平移和伸缩 \[S+\boldsymbol{a} = \{\boldsymbol{x+a}\arrowvert \boldsymbol{x}\in S\}, \alpha S=\{\alpha\boldsymbol{x}\arrowvert \boldsymbol{x}\in S\}\]
  2. 集合的和(不是集合的并): \(S_1+S_2=\{\boldsymbol{x+y}\arrowvert \boldsymbol{x}\in S_1, \boldsymbol{y}\in S_2\}\)
  3. 笛卡尔(Cartesian)积: \(S_1\times S_2 = \{(\boldsymbol{x_1, x_2})\arrowvert \boldsymbol{x_1}\in S_1,\boldsymbol{x_2}\in S_2 \}\)

线性分式及透视函数

透视函数

\[P:\bf{R}^{n+1}\rightarrow\bf{R}^n, P((\boldsymbol{z}, t)) = \boldsymbol{z}/t\] 对向量进行伸缩, 使得最后一个分量为 1. \(\bf{dom} P = \bf{R}^n\times \bf{R}_{++}\)

如果一个集合 \(C\subseteq \bf{dom}\ P\) 是凸的. 那么它的透射也是凸的. 可以证明线段的透射还是线段. 假设 \(\boldsymbol{x}=(\boldsymbol{\widetilde{x}}, x_{n+1}), \boldsymbol{y}=(\boldsymbol{\widetilde{y}}, y_{n+1}), x_{n+1} > 0, y_{n+1}>0\). 对于 \(0\leqslant\theta\leqslant 1\) \[P(\theta x+(1-\theta)y)=\dfrac{\theta\widetilde{x}+(1-\theta)\widetilde{y}}{\theta x_{n+1}+(1-\theta)y_{n+1}}=\mu P(x)+(1-\mu)P(y)\] \[\mu = \dfrac{\theta x_{n+1}}{\theta x_{n+1}+(1-\theta)y_{n+1}}\in [0,1]\] 可以看出 \(\mu\) 与 \(\theta\) 单调变换. 所以一个线段在透视函数下仍然是线段. 一个属于凸集的线段, 其象也是线段且属于凸集的象. 该象是凸的. 一个凸集在透视函数下的原象也是凸的
