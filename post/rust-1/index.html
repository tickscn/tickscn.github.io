变量(Variables)

虽然叫变量, 不过rust中变量默认是不可变(inmutable)的. 更安全和容易并发(concurrent)

如果确实需要更改变量, 可以在变量名前添加 mut

1
2
3
4
5
6

fn main() {
    let x = 5;
    let mut y = 5;
    x = 6; // error
    y = 6; // OK
}

不可变变量(Variables)与常量(Constants)的区别

  • 常量不可以使用 mut. 常量不是默认不可变, 而是一直不可变
  • 常量使用关键字 const 来声明, 并且需要指明类型
  • 常量可以存任何区域声明, 包括全局作用域
  • 常量只能设置为常量表达式, 不能是在程序运行期间运算的结果
1

const MAX_SIZE: u32 = 10_000

遮蔽(shadowing)

可以定义一个与之前同名的变量, 新的变量会屏蔽之前的变量. 使用 let

1
2
3
4
5
6

fn main(){
    let x = 5;
    let x = x + 1;
    let x = x * 2;
    println!("x = {}", x);
}

与 mut 不同, 使用 let 遮蔽后变量仍然默是不可变的

mut 不能修改变量的类型
