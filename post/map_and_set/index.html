pair

pair 定义在头文件 utility, 成员变量为 pubic, 分别为 first, second

  1. pair<T1, T2> p;
  2. pair<T1, T2> p(v1, v2);
  3. pair<T1, T2> p = {v1, v2};
  4. make_pair(v1,v2)
