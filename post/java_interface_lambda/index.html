接口(interface)

带有接口的类必须实现接口中的方法

1
2
3
4

public interface Comparable
{
    int compareTo(Object other);
}

类实现一个接口

  • 声明类的时候使用关键字 implements
  • 对接口中的方法进行定义

接口特性

  • 接口不是类
  • 接口变量必须引用实现了该接口的类对象
  • 可以使用 instanceof 检查对象是否实现了接口
  • 接口中的方法自动为 public, 接口中的域被自动设为 public static final
  • 可包含常量

静态方法

Java SE 8 中可以为接口增加静态方法

默认方法冲突

  1. 超类优先
  2. 接口冲突. 必须覆盖这个方法解决冲突

克隆

Cloneable 提示类实现了安全的克隆方式 clone Java 中赋值只是引用同的对象默认的拷贝方法 copy() 相当于 C++ 默认赋值

lambda 表达式

类似于匿名函数

方法引用

  • object::instanceMethod

  • Class::staticMethod

  • Class::instanceMethod

  • 前两种等价于提供参数的 lambda 表达式

  • 第三种方法, 第一个参数会成为方法的目标

变量作用域

lambda 表达式的三个部分

  1. 参数
  2. 代码块
  3. 自由变量的值, 指非参数且不在代码块中定义的值

自由变量可被 lambda 表达式捕获. 该变量必须是 final 的, 不能表达式中和表达式外改变

内部类包含一个引式引用, 可以访问外部类对象的私有域
