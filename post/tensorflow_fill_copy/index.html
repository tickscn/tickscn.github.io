填充

因为输入数据的原始值可能有各不相同的长度, 为了方便并行计算, 将不同的长度扩展为相同的长度. 之前可以通过 tf.tile 复制数据, 但这会破坏原有的数据结构, 通常应该是在需要补充长度的地方填充 0. 这时使用 ~tf.pad(x, [leftp, rightp]) 用 List 表示左右填充的数目

1
2
3

x = tf.constant([1, 2, 3])  # [1,2,3]
x = tf.pad(x, [0, 2])  # [1,2,3,0,0]
x = tf.pad(x, [2, 0])  # [0, 0, 1,2,3,0,0]

复制

并不是想要填充 0, 而是将已有数据复制. 使用之前介绍过和 tf.tile

数据限幅

  1. 下限幅: tf.maximum(x, a) 限制为 \([a, \infty)\) ReLU 可以通过 tf.maximum(x, 0) 实现
  2. 上限幅: tf.minimun(x,a) 限制为 \((-\infty, a]\)
  3. 同时限制上下幅度: 可以组合使用上限幅和下限幅. 也可以使用 tf.clip_by_value
