一些重要的凸集的例子

  1. 空集 \(\emptyset\), 单点集, 全空间都是全空间的仿射子集, 都是凸集
  2. 任意直线是仿射的. 通过零点直线同时也是锥.
  3. 长度非 0 的线段是凸的但不是仿射的
  4. 一条射线是凸的, 不是仿射的. 如果射线和基点是零点, 它也是锥

超平面

定义在 \(\bf{R}^n\) 的点集 \(\{\boldsymbol{x}\arrowvert \boldsymbol{a^Tx}=b\}\), 其中 \(\boldsymbol{a\in R^n, a\neq 0}. b\in \bf{R}\)

  1. 解析的看, 可以将其看作一个非平凡线性方程的解空间. 所以它是仿射的
  2. 几何上看, 可以将其看作一个法线向量为 \(\boldsymbol{a}\) 的超平面

半空间

一个超平面可以将全空间划分为两个半空间. 表示为 \(\{\boldysmbol{x}\arrowvert \boldsymbol{a^Tx}\leqslant b\}\).

半平面是凸的, 但是不是仿射的

半平面 \(C\) 凸性的证明: \(x_1, x_2 \in C, 0\leqslant \theta \leqslant 1. a^T(\theta x_1 +(1-theta)x_2) = \theta a^Tx_1 + (1-\theta)a^Tx_2\leqslant \theta b + (1-theta)b=b\)

欧几里得(Euclid)球和椭球

\(\bf{R}^n\) 空间中的 Euclid 球定义为 \[B(\boldsymbol{x}_c, r)=\{\boldsymbol{x}\arrowvert\Vert \boldsymbol{x-x_c}\Vert_2\leqslant r\}=\{\boldsymbol{x}\arrowvert \boldsymbol{(x-x_c)^T(x-x_c)}\leqslant r^2\}\] 其中 \(\Vert\cdot\Vert_2\) 指 Euclid 范数. 称 \(x_c\) 为球心, \(r\) 为半径. 它是凸集

一类相关的凸集为椭球 \[\epsilon = \{\boldsymbol{x}\arrowvert \boldsymbol{(x-x_c)^P^{-1}(x-x_c)}\leqslant 1\}\] 其中 \(P=P^T\succ 0\) 是对称正定矩阵. P 的特征值 \(\lambda_i\) 决定了半长轴

范数球和范数锥

一般的范数 \(\Vert\cdot\Vert\), 定义的范数球 \(\{\boldsymbol{x}\arrowvert \Vert\boldsymbol{x-x_c}\Vert\leqslant r\}\) 是凸的

多面体

多面体是有限个线性不等式和等式的解集. 是有限个半空间和超平面的交集 \[\mathcal{P}=\{\boldsymbol{x}\arrowvert \boldsymbol{Ax\preceq b, Cx=d}\}\] \(\preceq\) 表示 向量不等式 或 分量不等式

单纯形

单纯形是一类多面体. 设 k+1 个点 \(\boldsymbol{v_0, v_1, \cdots, v_k}\) 仿射独立. 它们决定了一个单纯形 \[C=\bf{conv}\{\boldsymbol{v_0,v_1,\cdots,v_k}\}=\{\theta_0v_0+\theta_2v_2+\cdots+\theta_kv_k\arrowvert \boldsymbol{\theta\succeq 0, 1^T\theta}=1\}\]

仿射独立类似于线性独立, 不存在不全为 0 的 \(a_1+a_2+\cdots+a_k=0\) 且 \(a_1v_1+a_2v_2+\cdots+a_kv_k=0\). 称 \(v_1, v_2, \cdots, v_k\) 仿射独立

\(\boldsymbol{x}\in C\) 的充要条件是存在 \(\boldsymbol{\theta\succeq 0, 1^T\theta}=1\) 使得 \(\boldsymbol{x}=\theta_0\boldsymbol{v}_0+\theta_1\boldsymbol{v}_1+\cdots+\theta_k\boldsymbol{v}_k\)

等价描述. 定义 \(\boldsymbol{y}=(\theta_1, \theta_2,\cdots,\theta_k)\) 和 \(B=[\boldsymbol{v_1-v_0, v_2-v_0,\cdots,v_k-v_0}]\in \bf{R}^{n\times k}\). \(\boldsymbol{x}\in C\) 的充要条件件是 \[\boldsymbol{x=v_0+By, y\succeq 0, 1^Ty}\leqslant 1\] B 的秩是 k, 所以存在非奇异矩阵 \(\boldsymbol{A=(A_1, A_2)}\in \bf{R}^{n\times n}\) 使得 \[\boldysmbol{AB=(A_1, A_2)B=(I,0)}\] 所以充要条件是 \[\boldysmbol{A_2x=A_2v_0, A_1x\succeq A_1v_0, 1^TA_1x\leqslant 1^TA_1v_0}+1\]

半正定锥

用 \(\bf{S}^n\) 表示 \(n\times n\) 的对称阵的集合. \(\bf{S}^n = \{\boldsymbol{A}\in \bf{R}^{n\times n}\arrowvert \boldsymbol{A=A}^T\}\). \(\bf{S}_+\) 半正定对称矩阵集合. \(\bf{S}_{++}\) 正定矩阵集合

$\bf{S}_+^n$是一个凸锥
