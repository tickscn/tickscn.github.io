使用

  • 汇编一个文件，输出为 a.out: as 文件.s
  • 汇编文件，并指定输出文件: as 文件.s -o 输出.o
  • 通过跳过空格和注释的预处理过程来更快的产生输出文件（只应该用于可信任的编译器的输出）: as -f 文件.s
  • 将给定路径添加到目录列表，来搜索.include 指令指定的文件: as -I 目录路径 文件.s 更复杂的操作可以看 man page

// demo.c
#include <stdio.h>

void fnu1()
{
    int i, j = 0;
    for (i = 0; i < 10000; ++i) j += i;
}

void fnu2()
{
    int i, j;
    fnu1();
    for (i = 0; i < 20000; ++i) j = i;
}

int main(void)
{
    int i;
    for (i = 0; i < 100; ++i) fun1();
    for (i = 0; i < 50000; ++i) fun2();
    return 0;
}
