Go 语言中有三种类型的函数, 具有名字的普通函数, 匿名函数或 lambda 函数, 方法. 除了 main 和 init 函数外其它函数都可以有参数和返回值. Go 中强制要求左大括号不换行

函数参数与返回值

Go 语言支持多返回值, 声明有返回值的函数需要由 return 和 panic, 在 Go 语言中每个代码分支都要有. C/C++中只是警告

按值与按引用传递

Go 默认使用按值传递, 如果想要操作实参要将参数的地址传入(仍然是按值传递). 切片, Map, 数组, 接口, 通道都是默认传递指针

package main

import (
  "fmt"
  "math/rand"
  "os"
  "time"
)

func IntPow(x, y int) (res int, err error) {
  if x == 0 && y != 0 {
    return
  }
  if x == 0 && y == 0 {
    err = fmt.Errorf("x, y 同时为0")
    return
  }
  res = 1
  for y != 0 {
    if y&1 != 0 {
      res *= x
    }
    x *= x
    y >>= 1
  }
  return
}

func main() {
  rand.Seed(time.Now().Unix())
  a := rand.Intn(101)
  b := rand.Intn(11)
  c, err := IntPow(a, b)
  if err != nil {
    fmt.Fprintln(os.Stderr, err)
  }
  fmt.Printf("%d^%d=%d\n", a, b, c)
}

传递变长参数

函数最后一个参数是 ...type 的形式, 可处理一个变长参数, 参数数目可以是 0. 可以使用类似于切片的语法访问变长参数, 也可以通过 slice... 的形式将一个切片传递给变长参数

package main

import "fmt"

func varages(s ...string) {
  if len(s) == 0 {
    return
  }
  fmt.Println(s[0])
  varages(s[1:]...)
}
func main() {
  s := []string{"Hello", "asdwda", "sdawete", "dawd"}
  varages(s...)
}

defer 和追踪

关键字 defer 可以将语句推迟到函数立马结束时执行(如 return 之后). 当有多个 defer 语句时它们按照注册顺序的逆序执行. defer 常用来

  1. 关闭资源(如文件流)

  2. 解锁

  3. 打印最终报告

package main

import "fmt"

func Adder() func(int) int {
  var x int
  return func(i int) int {
    x += i
    return x
  }
}

func main() {
  f := Adder()
  g := Adder()
  fmt.Println(f(1), " ", f(10), " ", f(100)) // 1, 11, 111
  fmt.Println(f(1), " ", g(10), " ", f(100)) // 112, 10, 212
}

使用闭包调试

对于调试, 知道具体运行的函数是很有用的, 可以使用 runtime 或 log 包中的函数来实现. 包 runtime 中的 Caller 可以提供一些有用的信息. 可以在相应的地方产生一个闭包来记录信息

package main

import (
  "log"
  "runtime"
)

func main() {
  where := func() {
    _, file, line, _ := runtime.Caller(1)
    log.Printf("%s:%d", file, line)
  }
  where()
  where()
}
