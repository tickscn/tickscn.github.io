类, 超类, 子类

定义子类

Java 使用 关键字 extends 来继承. Java 只有公有继承. 没有私有和保护继承

覆盖方法

子类的方法不能访问超类的私有域 super 可以告诉编译器调用超类的方法

声明为超类的变量既可以引用超类, 也可以引用子类. Java 默认为动态绑定, 不需要声明为虚拟方法

继承层次

一个类到其祖先的继承路径称为继承链

多态

Java 中对象变量是多态的, 既可以引用本类对象, 也可以引用其子类

阻止继承: final 类和方法

  1. 用 final 修饰的类不可扩展
  2. final 修饰的方法不可以被覆盖
  3. final 类的方法默认为 final 方法
  4. final 类的域不会默认为 final

强制类型转换

(type-id) val. 将 val 转为 type-id 的类型判断是否可以转换 instanceof

抽象类

abstract 含有抽象方法的类必须声明为抽象类. 抽象方法类似于 C++ 中的纯虚函数

抽象类不能实例化

受保护访问

  • private 本类可以访问
  • public 访问不受限制
  • protected 本包和所有子类可见
  • 本包可见, 默认

对象包装器与自动装箱

基本数据类型都有与之相对的对象 Byte, Short, Integer, Long, Float, Double, Character, Void, Boolean 这些对象包装器是不可变的, 且是 final 的, 不可派生
