back_insert

插入迭代器(insert iterator)是用来向容器中手插入元素的迭代器. 普通迭代器赋值时会写入当前迭代器所标志的位置, 但是赋值给插入迭代器会写入右边的新元素. 为了解释如何使用写入算法, 使用 iterator 中定义的函数 back_inserter

#include <vector>
#include <algorithm>
#include <iostream>
#include <iterator>
#include <numberic>
using namespace std;
int main()
{
    vector<int> vec;
    auto it = back_inserter(vec);
    *it     = 42;
    for (auto v : vec) cout << v << endl;
    return 0;
}

inline bool compareBySize(string& s1, string& s2)

lambda 表达式

可以应用调用操作符的对象或表达式是可调用的.

  • 函数
  • 函数指针
  • 重载了()运算符的类对象
  • lambda 表达式

lambda 表达式代表可调用代码单元, 可以认为是匿名的内联函数, 有返回值, 参数列表, 函数体

    sort(vec.begin(), vec.end());
    stable_sort(vec.begin(), vec.end(), [](const string& s1, const string& s2) { return s1.size() < s2.size(); });
    for_each(vec.begin(), vec.end(), [](const string& s) { cout << s << endl; });
    auto bound = partition(vec.begin(), vec.end(), [sz](const string& s) { return s.size() >= sz; });
    cout << "--------------------------------" << endl;
    for_each(vec.begin(), bound, [](const string& s) { cout << s << endl; });
    return 0;
}

istream_iterator 可以使用延迟加载. 将其绑定到流时, 并不保证立即读取流. 读取可以延迟到我们使用该迭代器为止. 一般情况下立即读取与延迟读取结果没有区别, 不过当一个流在读取之前被销毁或不同的对象同时读取时, 读取操作需要非常小心

反向迭代器(reverse iterator)

返向迭代容器, 除了 forward_list, 容器都有反向迭代器. rbegin, rend, crbegin, crend 可以产生反向迭代器

普通算法的结构

算法最重要的一点是其要求的迭代器支持的操作, 例 find 只需要访问, 对比是否相等, 增加迭代器. 而 sort 需要随机访问, 读, 写, 比较.

5 种迭代器类别

  • Input iterator
  • Output iterator
  • Forword iterator
  • Bidirectional iterator
  • Random-access iterator

参数匹配

  1. alg(beg, end, other args)
  2. alg(beg, end, dst, other args) : 需要保证 dst 有足够的空间
  3. alg(beg, end, beg2, other args)
  4. alg(beg, end, beg2, end2, other args)

特殊容器算法

list 包括 list, forward_list 有自己的 sort, merge, remove, reverse, unique
