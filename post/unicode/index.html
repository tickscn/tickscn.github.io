为了制定一个适应多语言的统一编码系统, 多个机构成立了 Unicode 联盟. 2019 年 5 月发布的 Unicode 12.1 包含 137994 个字符, 共 150 种语言

统一字符集(Universal Coded Character Set, UCS)

通用字符集（Universal Character Set）是由 ISO 制定的 ISO 10646（或称 ISO/IEC 10646）标准所定义的标准字符集, 与 Unicode 兼容. 定义了 0x10FFFF 个码点(code point)

Unicode 转换格式(Unicode Transformation Format, UTF)

现在常用有 UTF 有 UTF-8, UTF-16, UTF-32. 其中 UTF-8, UTF-16 是变长编码外 UTF-8 成为现时互联网上最流行的格式, 有几个原因:

  • 兼容 ASCII
  • 没有字节序问题

基本多文种平面(basic multilingual plane, BMP)

U+0000 到 U+FFFF 的字符被称为基本多文种平面

UTF-8

基本多文种平面(basic multilingual plane, BMP)

U+0000 到 U+FFFF 的字符被称为基本多文种平面

  1. US-ASCII 只用一个字节编码(U+0000 到 U+007F)
  2. 带有符号的拉丁文, 希腊文, 等字母用两个字符编码(U+0080 到 U+07FF)
  3. 其它 BMP 字符(大部分汉字位于这里) 用三个字符编码(U+0800 到 U+FFFF)

使用辅助平面的字符

  • 代理对
    • 高 U+D800 到 U+DBFF
    • 低 U+DC00 到 U+DFFF
