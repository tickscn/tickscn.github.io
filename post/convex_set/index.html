仿射集合和凸集

直线与线段

在 \(\bf{R}^n\) 空间中的两个点 \(\boldsymbol{x_1, x_2}\), 那所有潢足以下形式的所有点 \[\boldsymbol{y} = \theta\boldsymbol{x_1} + (1-\theta)\boldsymbol{x_2}\] 组成一条穿越 \(\boldsymbol{x_1, x_2}\) 的直线, 当 $0\leqslant θ \leqslant 1$时构成了一条以 \(\boldsymbol{x_1, x_2}\) 为端点的的闭线段

仿射变换

如果对通过集合 \(C\subseteq \bf{R}^n\) 中任意两个点的直线上的所有点都在集合 \(C\) 中, 称集合 \(C\) 是仿射的, 即 \[\{C|\forcall \boldsymbol{x_1, x_2} \in C, \theta\in \bf{R}\Rightarrow \theta \boldsymbol{x_1} + (1-\theta)\boldsymbol{x_2}\in C\}\] 若 \(\theta_1+\theta_2+\cdots+\theta_k=1, \boldsymbol{x_1,x_2,\cdots.x_k}\in C\) 则定义 \(\theta_1\boldsymbol{x_1}+\theta_2\boldsymbol{x_2}+\cdots+\theta_k\boldsymbol{x_k}\) 为仿射组合. 仿射集合的定义即是包仿其中任意两点的仿射组合. 可以证明, 仿射集合包含其中任意点的仿射组合

线性方程 \(\boldsymbol{Ax=b}\) 的解空间 C 是一个仿射, 一个仿射也可以表示为线性方程的解空间. 其子空间是 \(\boldsymbol{Ax=0}\) 的解空间, 即 \(\boldsymbol{A}\) 的零空间. 证明:

设 \(x_1, x_2 \in C\), 即 \(Ax_1=Ax_2=b\). 那么 \(A(\theta x_1+(1-\theta)x_2)=\theta Ax_1+(1-\theta)Ax_2=\theta b+(1-\theta)b=b\). 所以 \(\theta x_1+(1-\theta)x_2\in C\)

称任意集合中所有点的所有仿射组合组成的集合为 C 的仿射包, 记为 \(\bf{aff} C\)

仿射维数与相对内部

  1. 集合 C 的仿射维数是其仿射包的维数. 它与其他维数定义常常不相容. 如 \(\bf{R}^2\) 上的单位圆环 \(\{\boldsymbol{\boldsymbol{x\in R}^2\arrowvert x_1^2+x_2^2=1}\}\), 其仿射包是 \(\bf{R}^2\), 所以维数是 2. 但是在大多数维度的定义下, 其维度为 1
  2. 如果对于集合 \(C\subseteq \bf{R}^n\) 的仿射维度小于 n, 那么它的仿射包 \(\bf{aff}\ C\) 中可以定义相对内部 \[\bf{relint}\ C=\{\boldsymbol{x}\in C | B(\boldsymbol{x}, r)\cap \bf{aff}\ C\subseteq C\ for \ some\ r>0\}\] 其中 \(B(\boldsymbol{x}, r)=\{\boldsymbol{y}|\Vert\boldsymbol{y-x}\Vert \leqslant r\}\) 是空间中的一个球, 如果一个点的邻域与 \(\bf{aff}\ C\) 的交集全部属于 \(C\), 那么这个点属于 \(C\) 的相对内部. 如果只有一部分交集属于 \(C\), 这个点属于 相对边界 记为 \(\bf{cl}\ C \ \bf{relint}\ C\), \(\bf{cl}\ C\) 是 C 的闭包

凸集

当集合 \(C\) 中任意两个点的线段上的所有点都在集合 \(C\) 中, 定义它为凸集 \[C = \{\boldsymbol{x}\arrowvert \boldsymbol{x}= \theta\boldsymbol{x_1} + (1-\theta)\boldsymbol{x_2}\in C, \boldsymbol{x_1, x_2}\in C, 0\leqslant\theta\leqslant 1\}\] 根据定义仿射集是凸集, 对比仿射组合, 可以定义凸组合. 凸组合可以看作点的 加权平均. 集合 C 中所有点的凸组合为其凸包, 记为 \(\bf{conv}\ C\) 如果 \(B\) 是凸集且 \(C \subseteq B\), 那么 \(\bf{conv}\ C\subseteq B\)

可以自然的推论到函数 \(p:\bf{R}^n\rightarrow\bf{R}\) 如果 \(p(\boldsymbol{x})\geqslant 0\) 且 \(\int_C p(\boldsymbol{x})\mathrm{d}\boldsymbol{x}=1\), 其中 C 是凸集. 如果积分 \(\boldsymbol{y}=\int_C \boldsymbol{x}p(\boldsymbol{x})\mathrm{d}\boldsymbol{x}\) 存在, 那么 \(\boldsymbol{y}\in C\)

锥

锥又被称为非负齐次. 即满足 \(\forall \boldsymbol{x}\in C, \theta\geqslant 0\Rightarrow \theta\boldsymbol{x}\in C\) 的集合 \(C\) 可类比凸组合与仿射组合给出 锥组合 与 锥包 的定义
