继承

类之间通过层级结构继承相关联. 层级结构的根是基类. 其它类直接或间接的继承于它. 继承的类被称为衍生类(derived class). 基类定义层级中通用的成员. 继承类定义它们自己特殊的成员. C++ 中, 区分基类与衍生类的相同函数时, 默认认为两者相同, 若衍生类要定义它自己的版本, 基类函数声明为 virtual. 继承时通过冒号后的起源类列表来表明继承关系. 每个基类有一个可选的访问修饰符

    std::string isbn() const;
    virtual double net_price(std::size_t n) const;

class Bulk_Quote : public Quote {
public:
    double net_price(std::size_t n) const override;

class Quote {
public:
    Quote() = default;
    Quote(const std::string& book, double sales_price) : bookNo(book), price(sales_price) {}
    std::string isbn() const
    {
        return bookNo;
    }
    virtual double net_price(std::size_t n) const
    {
        return n * price;
    }
    virtual ~Quote() = default;
private:
    std::String bookNo;
protected:
    double price = 0.0;
};

成员函数和继承

衍生类继承基类的成员, 但是衍生类有时需要定义它自己的操作定义. 此时衍生类通过提供自己的实现来覆盖继承的实现. c++ 中基类需要区分希望衍生类原样继承或希望衍生类提供自己实现的类. 使用 virtual 表明希望被覆盖. 非静态, 非构造函数都可以是 virtual. 只需要声明时添加该关键字就可以

虚函数

动态绑定发生在基类的指针和引用调用虚函数时. 我们必须定义所有的虚函数, 而普通的函数在不用的时候可以不定义

虚函数调用 可能 在运行时重新确定

动态绑定时编译器生成代码来决定哪个函数被使用.

在衍生类中的虚函数

一但一个函数被声明为 virtual, 所有的衍生类中都是 virtual. 覆写的函数必须有相同的名字和参数列表. 返回值也应相同, 例外是返回的是自身的引用或指针

虚函数与默认参数

如果基类与衍生类的虚函数默认参数不同, 即使绑定的是衍生类的基类指针仍然会调用基类函数

#include <iostream>
#include <vector>
using namespace std;
class A {
public:
    virtual void test()
    {
        cout << "A" << endl;
    }
};

class B : public A {
public:
    void test()
    {
        cout << "B" << endl;
    }
};

int main(int argc, char* argv[])
{
    A a;
    B b;
    A& ra = a;
    A& rb = b;
    ra.test();
    rb.test();
    vector<A> vec;
    vec.push_back(ra);
    vec.push_back(rb);
    for (auto c : vec) c.test();
    return 0;
}
