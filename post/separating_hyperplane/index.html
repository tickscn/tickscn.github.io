分离与支撑超平面

分离超平面

两个不相交的凸集可以用一个超平面或仿射函数分离开来假设凸集 \(C\) 和 \(D\) 且 \(C\cap D=\emptyset\), 那么存在 \(\boldsymbol{a\neq 0}, b\) 使得 \(\forall\boldysmbol{x}\in C, \boldsymbol{a^Tx}\leqslant b. \forall\boldsymbol{x}\in D, \boldsymbol{a^Tx}\geqslant b\). 超平面 \(\{\boldsymbol{x\arrowvert a^Tx}=b\}\) 被称为集合 C,D 的分离超平面

线性空间中一条直线有界, 说明其系数为 0

严格分离

要求凸集与超平面没有交点

超平面分离定理的逆定理

如果没有更多约束, 超平面的逆定理是不成立的. 只要两个凸集的交集在一条直线上. 添加一些条件的逆定理

如果两个凸集 C 和 D. 其中一个是开集. 如果存在一个仿射函数在 C 中非正, 在 D 中非负. 那么 C 和 D 不相交

  1. \(\boldsymbol{Ax}\prec \boldsymbol{b}\)
  2. \(\begin{cases}\boldsymbol{\lambda} &\neq \boldymbol{0}\ \boldsymbol{\lambda} &\succeq \boldsymbol{0}\ \boldsymbol{A^T\lambda} &= \boldsymbol{0} \\\
    \boldsymbol{\lambda^Tb}&\leqslant 0\end{cases}\)

上述两个方程只能有一个有解

支撑超平面

设集合 \(C\subseteq \bf{R}^n\), 存在边界上一点 \[\boldsymbol{x_0}\in \bf{bd}\ C=\bf{cl}\ C\smallsetminus \bf{int}\ C\] 如果 \(\exists\boldsymbol{a}\) 使得 \(\forall\boldsymbol{x}\in C, \boldsymbol{a^Tx\leqslant a^Tx_0}\) 称超平面 \(\{\boldsymbol{x}\arrowvert \boldsymbol{a^Tx=a^Tx_0}\}\) 为集合 \(C\) 在点 \(\boldsymbol{x_0}\) 处的支撑超平面
