套接字(Socket)是一个抽象概念, 负则启动通信. 接下来介绍仅包含 TCP (传输控制协议)

  • Socket(String host, int port)
  • InputStream getInputStream()
  • OutputStream getOutputStream()

套接字超时

读写超时

从套接字读取信息时, 在有数据可访问之前, 读操作会阻塞. 为了不因为主机不可达而等待过长时间, 可以设置超时值

  • SetSoTimeout

如果超时会抛出 SocketTimeoutException

Socket s = new Socket();

因特网地址

  • IPV4 中是以 4 个字节组成
  • IPV6 中是以 16 个字节组成

通常不用过多考虑因特网地址的问题. 但是, 如果需要主机名和因特网地址之间进行转换, 可以使用 InetAddress 类

  • InetAddress address = InetAddress.getByName(host)

只得到 localhost 地址, 总会得到本地回环地址 127.0.0.1, 但是其它程序无法使用该地址到这台计算机. 可以使用静态方法 getLocalHost 获取本地主机地址

可中断套接字

连接和读写都有可能阻塞当前线程. 为了可以中取消那些很有可能没有结果的连接, 可以使用 java.nio 包提供的 SocketChannel 类 SocketChannel channel = SocketChannel.open(new InetSocketAddress(host, port)) . channel 提供了 read 和 write 方法

获取 Web 数据

URL 和 URI

URL 和 URLConnection 类封装了大量细节. 可以通过字符串构建一个 URL.

1
2

URL url = new URL(urlString);
Scanner in = new Scanner(url.openStream(), "UTF-8");

URL(Uniform Resource Locater, URL) 统一资源定位符是 URI 的特例, 不能定位资源的 URI 被称为统一资源名称(Uniform Resource Name, URN)

URI 类只负责解析, URL 可以打开一个到达资的源. URL 只能用于 JAVA 知道如何处理的模式, 如 http:, https:, ftp:, file:, jar:

URI 语法

[schema:]schemaSpecialPart[#fragment]

  • […]表示可选部分
  • 包含 schema: 部分的 URI 为绝对 URI, 否则为相对 URI
  • schemaSpecialPart 不以 / 开头. 称其为不透明的
  • 绝对的透明 URI 和所有的相对 URI 都是分层的
    • 分层结构 [//authority][path][?query]
    • authority: [user-info@]host[:port]

URI 类的作用是解析标识符, 可以使用 getXXX 获取

使用 URLConnection

URLConnection 比 URL 有更多的控制功能

  1. 调用 URL 的 openConnection 获取 URLConnection =URLConnection connection = url.openConnection()
  2. 设置请求属性
    1. setDoInput
    2. setDoOutput
    3. setIfModifiedSince
    4. setUseCaches
    5. setAllowUserInteraction
    6. setRequestProperty
    7. setConnectionTimeout
    8. setReadTimeout
  3. 调用 connect 方法连接远程资源 connection.connect()
  4. 获取资源信息
    1. getContentType
    2. getContentLength
    3. getContentEncoding
    4. getDate
    5. getExpiration
    6. getLastModified
  5. 获取输入流 getInputStream

默认只产生输入流, 如果想要向服务器提交数据, 需要设置 connection.setDoOutput(true)

提交表单数据

有许多技术可以让 Web 服务器调用程序, 如 Java Servlet, JavaServer Face, ASP(Active Server Pages), CGI(Common Gateway Interface) 以下主要讨论客户端如何与以知的服务端交互. 当向 Web 服务器发送信息时, 常用的两个命令: GET 和 POST

GET

将参数附加再 URL 结尾处: http://host/path?query. 每个参数都具有"名字=值"的形式. 参数用&隔开. 值将遵循下面的规则

  • 保留 a-z, A-Z, 0-9, 以及 . - ~ _
  • 用 + 替换所有空格
  • 将其它所有字符编码为 utf-8, 并将每个字节写为 % 后面跟两个 16 进制数字

POST

为了处理具有大量数据的表单. 可以使用 POST. POST 并不在 URL 上附着参数, 从 URLConnection 获取输出流, 并将名, 值对写入该流.

package post;

import java.io.*;
import java.net.*;
import java.nio.file.*;
import java.util.*;

/**
 * This program demonstrates how to use the URLConnection class for a POST request.
 * @version 1.40 2016-04-24
 * @author Cay Horstmann
 */
public class PostTest
{
   public static void main(String[] args) throws IOException
   {
      String propsFilename = args.length > 0 ? args[0] : "post/post.properties";
      Properties props = new Properties();
      try (InputStream in = Files.newInputStream(Paths.get(propsFilename)))
      {
         props.load(in);
      }
      String urlString = props.remove("url").toString();
      Object userAgent = props.remove("User-Agent");
      Object redirects = props.remove("redirects");
      CookieHandler.setDefault(new CookieManager(null, CookiePolicy.ACCEPT_ALL));
      String result = doPost(new URL(urlString), props,
         userAgent == null ? null : userAgent.toString(),
         redirects == null ? -1 : Integer.parseInt(redirects.toString()));
      System.out.println(result);
   }

   /**
    * Do an HTTP POST.
    * @param url the URL to post to
    * @param nameValuePairs the query parameters
    * @param userAgent the user agent to use, or null for the default user agent
    * @param redirects the number of redirects to follow manually, or -1 for automatic redirects
    * @return the data returned from the server
    */
   public static String doPost(URL url, Map<Object, Object> nameValuePairs, String userAgent, int redirects)
         throws IOException
   {
      HttpURLConnection connection = (HttpURLConnection) url.openConnection();
      if (userAgent != null)
         connection.setRequestProperty("User-Agent", userAgent);

      if (redirects >= 0)
         connection.setInstanceFollowRedirects(false);

      connection.setDoOutput(true);

      try (PrintWriter out = new PrintWriter(connection.getOutputStream()))
      {
         boolean first = true;
         for (Map.Entry<Object, Object> pair : nameValuePairs.entrySet())
         {
            if (first) first = false;
            else out.print('&');
            String name = pair.getKey().toString();
            String value = pair.getValue().toString();
            out.print(name);
            out.print('=');
            out.print(URLEncoder.encode(value, "UTF-8"));
         }
      }
      String encoding = connection.getContentEncoding();
      if (encoding == null) encoding = "UTF-8";

      if (redirects > 0)
      {
         int responseCode = connection.getResponseCode();
         if (responseCode == HttpURLConnection.HTTP_MOVED_PERM
               || responseCode == HttpURLConnection.HTTP_MOVED_TEMP
               || responseCode == HttpURLConnection.HTTP_SEE_OTHER)
         {
            String location = connection.getHeaderField("Location");
            if (location != null)
            {
               URL base = connection.getURL();
               connection.disconnect();
               return doPost(new URL(base, location), nameValuePairs, userAgent, redirects - 1);
            }

         }
      }
      else if (redirects == 0)
      {
         throw new IOException("Too many redirects");
      }

      StringBuilder response = new StringBuilder();
      try (Scanner in = new Scanner(connection.getInputStream(), encoding))
      {
         while (in.hasNextLine())
         {
            response.append(in.nextLine());
            response.append("\n");
         }
      }
      catch (IOException e)
      {
         InputStream err = connection.getErrorStream();
         if (err == null) throw e;
         try (Scanner in = new Scanner(err))
         {
            response.append(in.nextLine());
            response.append("\n");
         }
      }

      return response.toString();
   }
}
