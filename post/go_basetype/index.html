Go 语言将数据类型分为四类:

  1. 基础类型
  2. 复合类型
  3. 引用类型
  4. 接口类型

整型

Go 语言提供了多种大小的有符号和无符号整型, 有符号整型包括 int8, int16, int32, int64, 前面加上 u 就可表示相应的无符号

还有针对特定 CPU 平台机器字定义的 int 和 uint 代表 32 位或 64 位整型

Unicode 字符码点 rune 是 int32 的等价类型, byte 是 uint8 的等价类型

还有用来表示指针的 uintptr

支持的算术运算符与 C 相比

  1. 多了一个 &^ 按位置 0
  2. 按位取反复用了 ^ 运算符. 移位运算符的优先级比
  3. 移位运算优先级提升, 只比乘除取模低

为避免优先级混淆, 建议使用括号

格式化打印时, 可以使用 %d, %o, %x 表示 10 进制, 9 进制, 16 进制. 使用非 10 进制时可以使用 \# 表示打印前缀, 使用 %c 打印字符, 使用 %p 打印带有引号的字符

浮点数

Go 定义两种浮点数 float32 和 float64, 分别相当于 C 语言中的 float 和 double

打印时%g 表示尽量紧凑的表示, %e 表示科学计数法, %f

复数

Go 提供了 complex64 和 complex128 分别对应由 float32 和 float64 组虚部和实部的复数. 可以用 complex 函数生成复数, real 函数返回实部, imag 返回虚部包 math/cmplx 提供了很多的复数处理函数

布尔

只有两种值 true 和 false. 与 C 不同, 非 0 值不会默认转变 false

字符串

字符串是不可改变的常序列

len 函数可以返回字符串 字节 的数目. 可以使用 [i:j] 操作符构建子字符串, 由原字符串的第 i 个 字节 到第 j 个 字节. 可以使用 + 拼接字符串. 字符串是可比较的

字符串字面值

使用双引号括起来的部分, 可以使用 “\” 进行转义

Unicode

一个通用 Unicode 码点的数据类型是 int32, 在 Go 语言中可以用 rune 类型. 用 rune 序列表示符文序列的编码方式是 UTF-32 或 UCS-4. 是一种定长编码, 有些浪费空间. 所以有了变长的 UTF-8, UTF-16

UTF-8

编码 码点
0xxxxxxx 0-127 ASCII
110xxxxx 10xxxxxx 128-2047
1110xxxx 10xxxxxx 10xxxxxx 2048-65535
11110xxx 10xxxxxx 10xxxxxx 10xxxxxx 65536-0x10FFFF

变长编码无法随机访问某一个字符. Go 语言的源文件使用 UTF-8 编码, Go 处理 UTF-8 文本的能力也很强. 在文本中可以使用 \u 转义来输入一些很难直接输入的字符. 为了快速的随机访问, 可以将 UTF-8 字符串转为 rune 序列一些 unicode/utf8 包中的函数. 计算字符串的 字符 数量 RuneCountInString. 在使用 range 时会自动解码 UTF-8 字符串. 可以将字符串显示转为 rune 序列

Byte 切片

标准库有四个重要的字符串处理包, 分别为 bytes, string, unicode, strconv 因为字符串是只读的, 大量修改字符串时可能使用 bytes.Buffer 是更好的选择. strconv 提供了一些类型与字符串这间的相互转换为了避免转换过程中不必要的内存分配, bytes 和 strings 包同时提供了一些函数

func comma(s string) string {
  var buf bytes.Buffer
  len := utf8.RuneCountInString(s)
  p := len % 3
  buf.WriteString(s[:p])
  for ; len > p; p += 3 {
    buf.WriteByte(',')
    buf.WriteString(s[p : p+3])
  }
  return buf.String()
}

字符串和数字的转换

由 strconv 包提供这些转换功能. Itoa 可以将整数转为 ASCII 序列. FormatInt 和 FormatUint 可以按不同的进制格式化数字. 可以使用 Atoi 或 ParseInt 来将字符串转为整数. 可以使用 fmt 包的函数来进行转换

常量

常量是编译期计算, 运行时不可更改

iota 常量生成器

从遇到 const 关键字时初始化为 0, 每次出现自动加 1

无类型常量

可以提供更高精度的算术运算(最低 256bit), 而且使用时不需要强制类型转换. 只有常量可以是无类型的
