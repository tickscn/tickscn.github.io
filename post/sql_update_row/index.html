插入行

按列位置插入行

1
2

INSERT INTO table_n
VALUES(v1,v2,v3,...,vn);

按列名插入行

1
2
3

INSERT INTO table_n
            (colunm1, column2, ..., columnN)
VALUES(v1, v2, ..., vN);

忽略的列可以自动插入默认值或空值, 如果没有默认值且不允许空值, 插入会出错. 可以用一条插入语句插入多行, VALUES 子句中用逗号分隔多个圆括号括起来的值

从另一表中插入

1
2

INSERT INTO table_n
      subquery;

子查询返回满足条件的行. 可以批指定列名

更新行

1
2
3

UPDATE table_n
   SET column_n = expr_Value
   [WHERE search_condition]

将历史书价格翻倍

1
2
3

UPDATE titles
   SET price = price * 2
 WHERE type = 'history'

更新多列, 使用逗号分隔 列名=值 对

删除行

1
2

DELETE FROM table_n
[WHERE search_condition]

省略 WHERE 删除所有行

1

=TRUNCATE TABLE table_n= 不是 SQL 标准, 但是大部分 DBMS 支持. 删除整个表时比 DELETE 快

指导原则

  • 除非确实打算删除或更新整个表, 否则不要使用不带 WHEWE 子句的 UPDATE 和 DELETE
  • 保证每个表都有主键
  • 在进行 UPDATE 和 DELETE 之前, 应使用 SELECT 和 WHERE 来检查 WHEWE 的过滤是否正确
  • 使用强制实施引用完整性的数据库
