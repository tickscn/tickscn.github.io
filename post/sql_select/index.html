使有 DISTINCT 消除重复行

  • 如果有多列, 则所有列的值的组合来决定重复行
  • 只返回一个空值
  • 有显示所有行的 ALL 关键字, 不过因为与默认行为相同, 很少使用
  • DISTINCT 需要排序, 会花费大量时间, 如无必要, 不要使用
1
2

SELECT DISTINCT state
    FROM authors;

使用 ORDER BY 排序行

1
2
3
4
5
6

SELECT au_fname AS "first name",
       au_lname AS "last name",
       city, state
  FROM authors
  ORDER BY au_lname ASC
           city DESC

ORDER BY 需要指定排序列, 该列不用是检索的列, 默认是 ASC 递增排序, 可以指定 DESC 递减排序, 升序时空串在前, 降序时空串在后

  • ORDER BY 允许用数字引用检索列中的列名
  • 子串排序 MYSQL 支持 substr(str, begin)
  • CASE 根据逻辑排序

CASE WHEN ... THEN... ELSE... END

使用 WHERE 筛选行

条件

  1. 比较
    1. 相等 =
    2. 不等 <>
    3. 小于 <
    4. 大于 >
    5. 小于等于 <=
    6. 大于等于 >=
  2. 模式匹配 LIKE
  3. 范围筛选 BETWEEN
  4. 列表筛选 IN
  5. 空值测试 IS NULL
1

WHERE 计算在 SELECT 之前, 所以无法引用别名. 不过可以通过 FROM 来使用别名

1
2
3

SELECT *
  FROM (SELECT sales AS copies_sold FROM title) AS ta
  WHERE copies_sold >= 10000;

逻辑运算

AND

  • 连接两个条件且只在两个条件都为真时返回真
AND 1 0 NULL
1 1 0 NULL
0 0 0 0
NULL NULL 0 NULL

OR

OR 1 0 NULL
1 1 1 1
0 1 0 NULL
NULL 1 NULL NULL

NOT

1 0
0 1
NULL NULL

优先级

NOT > AND > OR

LIKE 模式匹配

  • 只对字符串有用
  • 可以使用 NOT LIKE 否定条件

通配符

  1. % 匹配 0 个或多个字符 (类似于正则表达式的 .*)
  2. _ 匹配任意 1 个字符(类似于正则表达式的 .)

可以使用 ESCAPE 指定转义符

BETWEEN

test_column BETWEEN lval AND rval 等价于 (test_column >\equal lval) AND (test_column <\equal rval)

IN

test_column IN (val1, val2, val3) 等价于 (test_column = val1) OR (test_column = val2) OR (test_column = val3)
